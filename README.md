# How to use the scripts

1. Open command line interface 'cmd'
   1. Either: open `cmd.exe` and then go to you desired directory via ` cd <path> `
   2. Or: open your desired folder > right click in window > click on ` open in terminal `
2. Clone repository from github:
```bash
# download code from github
git clone <link>
```

3. Enter the directory (folder) you just cloned:
```bash
# download code from github
cd implement_nmm
```

4. Initialize python virtual environment and install packages

```bash
# create venv (virtual environment)
python -m venv venv

# activate venv
venv\Scripts\activate

# install packages from requirements.txt
pip install -r requirements.txt
```

5. Open the file 'run basic models.py' in you IDE

```bash
explorer . # works on Windows to open the current directory in file explorer

code . # works on Windows to open current directory in VS code
```

6. Configure or change `run basic models.py` to your liking

Currently, `Montbrio et al 2015` is replicated in `run basic models.py`
